# 🍔 The Food Lab

The Food Lab is a responsive food ordering website built using **HTML, CSS, and JavaScript**.  
It provides an attractive user interface for browsing food items, viewing bestsellers, managing a shopping cart, and learning more about the restaurant.

---

## 🚀 Features

### 🏠 Home Page
- Hero section with background video
- Attractive landing page UI
- Bestseller food items showcase
- Navigation bar with search functionality

### 🛒 Cart System
- Add items to cart
- Increase/decrease quantity
- Remove items from cart
- Automatic total price calculation
- Discount system based on total amount
- Checkout functionality
- Cart data stored using Local Storage

### 📖 About Us Page
- Customer testimonials slider
- Animated counters
- Reveal-on-scroll animations
- Vision and story section
- Feature cards with hover effects

### 📞 Contact Us Page
- Contact information section
- Animated moving banner
- Responsive layout

### 🔍 Search Functionality
- Real-time food item filtering

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
- Intersection Observer API
